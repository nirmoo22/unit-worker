## Setting up on your machine

### Installing

From your command line go to your projects directory and type these commands
```bash
    git clone https://github.com/nirmoo22/unit-worker.git
    cd unit-worker
    npm install
```

### Run the server

```bash
    npm start
```

It should be running on https://localhost:3000
